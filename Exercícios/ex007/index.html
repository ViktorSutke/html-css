Tags Obsoletas vindas do HTML4

Em HTML4, colocamos o par de tags <u> e </u> para delimitar um termo.
Isso seria uma meneira de determinar somente um formato visual para chamar atenção.
O sublinhado é apenas uma forma, sem significado explícito.

Já a HTML5 possui o conceito de semântica, suas tags tentam levar um significadon embutido forte.
Usamos as tags <address> </address> para dar um significado ao destaque que fizemos.
